About Student Manager

Student Manager is a comprehensive system for managing student information and records.

Features:

  • User authentication with role-based access
  • Complete CRUD operations for student management
  • Admin and User role permissions
  • Responsive design for all devices
  • Modern and intuitive interface

User Roles:

Administrator

Full access to all features including create, read, update, and delete operations.

User

Read-only access to view student information and records.
